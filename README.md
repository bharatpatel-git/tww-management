# The Wild Woods 🌲

An internal hotel management system built with **React**, **Vite**, and **Supabase**.

> **Note**: I built this project while learning and practicing from the course **[The Ultimate React Course](https://www.udemy.com/course/the-ultimate-react-course/)** by Jonas Schmedtmann.

## Key Features

- **Dashboard**: Real-time overview of bookings, check-ins, and occupancy.
- **Management**: CRUD operations for Cabins, Guests, and Bookings.
- **Tech Stack**: React Query, Styled Components, React Hook Form, Date-fns.

## Quick Start

1. Run `npm install` to install dependencies.
2. Create a `.env` file based on `.env.example` and add your Supabase credentials.
3. Run `npm run dev` to start the app.

## Deployment

### GitHub

1. Create a new repository on GitHub.
2. Initialize git: `git init` (if not already done).
3. Add files: `git add .`
4. Commit: `git commit -m "Initial commit"`
5. Link to GitHub: `git remote add origin YOUR_REPOSITORY_URL`
6. Push: `git push -u origin main`

### Vercel

1. Connect your GitHub repository to Vercel.
2. Add the environment variables (`VITE_SUPABASE_URL` and `VITE_SUPABASE_KEY`) in the Vercel project settings.
3. Vercel will automatically detect Vite and deploy your project!
